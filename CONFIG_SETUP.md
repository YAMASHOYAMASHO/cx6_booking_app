# ⚠️ デプロイ前に必ず編集してください

## 🎯 編集が必要なファイル

### 📌 最重要: メールドメインの設定

**ファイルパス**: 
```
lib/src/config/auth_config.dart
```

**編集箇所**:
```dart
// 行番号: 約13行目
static const String defaultEmailDomain = 'univ.com';  // ← ここを編集
```

**変更内容**:
- `'univ.com'` を実際の大学のメールドメインに変更してください
- 例: `'yourschool.ac.jp'`、`'university.edu'` など

---

## 📖 詳細なガイド

詳しい設定方法は `DEPLOYMENT_GUIDE.md` を参照してください。

## 🚀 クイックスタート（開発環境）

現在のデフォルト設定（`univ.com`）のまま、テストユーザーで動作確認できます:

### テストユーザー例

**学生ユーザー**:
- ユーザーID: `123456` (学籍番号のみ入力)
- 実際のメール: `123456@univ.com` (自動変換)

**ゲストユーザー**:
- ユーザーID: `guest@example.com` (フルメールアドレス入力)
- 実際のメール: `guest@example.com` (そのまま使用)

**注意**: 本番環境にデプロイする前に必ず設定を変更してください！
