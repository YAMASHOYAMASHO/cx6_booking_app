# タイムラインビュー

## 📅 概要
タイムラインビューは、装置の空き状況と予約を視覚化するためのコアインターフェースです。

## 🖥️ UIデザイン
- **軸**:
    - **垂直**: 時間 (0:00 - 24:00)。
    - **水平**: 装置 (場所ごとにグループ化)。
- **グリッド**:
    - 1時間 = 高さ60px (設定可能)。
    - 15分単位のスナップ間隔。

## 🧩 コンポーネント
- `TimelineGrid`: 背景のグリッド線を描画します。
- `ReservationBar`: 個々の予約ブロックを描画します。
    - **色**: 自分の予約と他人の予約で異なる色を使用。
    - **インタラクション**: クリックして詳細表示/編集。
- `CurrentTimeLine`: 現在時刻を示す赤い線。

## 🔄 ロジック
- **データ取得**: 選択された日付と場所の予約を取得します。
- **レイアウト計算**:
    - `Top`: `(時 * 60) + (分 / 60 * 60)` ピクセル。
    - `Height`: `分単位の期間 / 60 * 60` ピクセル。
- **競合の可視化**: 重複する予約 (データの不整合がある場合) は、並べて表示または積み重ねて表示されます。

## 🚀 パフォーマンス
- **仮想化**: 装置数が多い場合、水平スクロールは仮想化されます。
- **再構築の最適化**: タイマーの更新時 (現在時刻線用) には、タイムライン領域のみが再構築されます。
